# Chess Game in Java

This is a Java-based chess game that allows two players to compete in a standard chess match. The game includes all standard chess rules, such as castling, en passant, pawn promotion, checkmate, and stalemate.

## Features

- Two-player mode
- Standard chess rules
- Castling
- En passant
- Pawn promotion
- Checkmate detection
- Stalemate detection
- Graphical user interface
